■ ランダム兵団編成ツール Readme

このアプリは、ゲーム「Conqueror's Blade（コンカラーズブレード）」の
兵団編成を補助するための非公式ツールです。
ゲーム運営会社および関連企業とは一切関係ありません。

----------------------------------------
■ 免責事項
----------------------------------------

・このツールの利用によって発生したいかなる損害についても、
  開発者は一切の責任を負いません。

・本ツールは非公式であり、ゲーム内データの正確性を保証するものではありません。

----------------------------------------
■ 著作権について
----------------------------------------

・ゲーム内の名称、データ、画像、その他の知的財産権は
  すべてゲーム運営会社に帰属します。

・本ツールのコード部分については、個人利用・改変は自由ですが、
  商用利用は禁止とします。
　再配布については開発者（Cheney）にご相談下さい

----------------------------------------
■ ファイル構成
----------------------------------------

main.exe（または main.py）
units.json
tutorial.txt
user_settings.json（自動生成）
Readme.txt

----------------------------------------
■ 注意
----------------------------------------

・user_settings.json はアプリ終了時に自動保存されます。
・units.json が存在しない場合、アプリは動作しません。

----------------------------------------
■ 開発者より
----------------------------------------

このツールは、兵団編成の幅を広げたり、
普段使わない兵団を試すきっかけになることを目的としています。

自由にカスタマイズして楽しんでください。
