# ランダム兵団編成ツール Readme

このアプリは、ゲーム「Conqueror's Blade（コンカラーズブレード）」の
兵団編成を補助するための非公式ツールです。
ゲーム運営会社および関連企業とは一切関係ありません。

----------------------------------------
■ 導入方法
----------------------------------------
 ・画面右側のreleaseからLatestを開きdefalt.zipをダウンロードしたのちに解凍しmain.exeを起動してください

----------------------------------------
■ ファイル構成
----------------------------------------

main.exe
units.json
tutorial.txt
user_settings.json（自動生成）
Readme.txt

----------------------------------------
■ 注意
----------------------------------------

・user_settings.json はアプリ終了時に自動保存されます。
・units.json が存在しない場合、アプリは動作しません。
